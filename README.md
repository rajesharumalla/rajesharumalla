## Project Spotlight

### Job Alerts Automation

A practical automation workflow that monitors fresher and entry-level software opportunities and sends structured alerts to Discord every 4 hours. This project is designed to reduce manual job searching by collecting, filtering, scoring, and deduplicating opportunities from multiple public job sources and company boards.

**Key capabilities:**

- Runs automatically every 4 hours using an n8n schedule trigger
- Fetches job data from remote job APIs, company job boards, and Google Jobs search through SerpAPI
- Normalizes different response formats into one consistent job schema
- Filters for fresher, junior, graduate, 0-1 year, and 0-2 year technical roles
- Scores jobs using rule-based matching for software, automation, QA, embedded, DevOps, and support roles
- Deduplicates previously sent jobs using workflow static data
- Sends clean Discord alerts with company, role, location, posted date, score, and apply link

This project highlights the software side of my profile: workflow design, API integration, JavaScript data transformation, automation logic, scheduled execution, and practical problem solving.

### Multi-Protocol Signal Emulator

An STM32F407-based embedded system built to capture, store, verify, clone, emulate, and retransmit signals across multiple short-range communication domains.

**Key capabilities:**

- IR signal capture and replay using timer input capture and a 38 kHz PWM carrier
- RF packet receive and transmit through CC1101 at 433.92 MHz
- RFID trusted-key storage and verification using PN532
- NFC UID/NDEF read, save, clone, and basic tag emulation
- Menu-driven interface with a 16x2 I2C LCD and 4x4 keypad
- Internal flash storage for saved IR, RF, NFC data, and trusted RFID keys

This project helped me strengthen embedded debugging, protocol understanding, peripheral integration, menu-state design, and end-to-end firmware development.
